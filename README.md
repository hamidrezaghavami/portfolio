# Minimalist Developer Portfolio

A clean, text-forward single-page portfolio designed for showcasing high-integrity backend systems and robust full-stack software engineering.

## 🚀 Key Features
- **Ambient Mouse-Tracking Glow:** Dynamic, reactive radial lighting following cursor coordinates.
- **Dynamic Scroll Background Shifts:** Context-aware canvas color transitions on scroll.
- **Fluid Grid System:** Fully optimized responsive breakpoints for mobile and desktop screens.

## 📁 Core Structure
- `index.html` — Vanilla ES6+ JavaScript, Tailwind CSS structure, and layout logic.
- `.gitignore` — Core operational tracking blocklist.
